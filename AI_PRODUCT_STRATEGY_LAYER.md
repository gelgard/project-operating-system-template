# AI Product Strategy Layer

## Template Strategy

This repository is not an end-user software product.
It is a reusable project operating system.

## Core Value

It allows a user to create new projects that differ in:
- product type
- technology stack
- architecture
- complexity
- deployment model
- team shape

while preserving the same disciplined operating model.

## Strategic Promise

A project created from this template must be able to inherit:
- the same architecture-first startup
- the same recovery-first discipline
- the same AI task execution system
- the same response format discipline
- the same update and synchronization logic

without inheriting project-specific content.
