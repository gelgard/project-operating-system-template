TEMPLATE REPO UPDATE COMMAND

Command:
"обнови template-repo"

Purpose:
Safely synchronize the universal template repository with methodology-level changes discovered in the current real project.

Use this command only when the template repository already exists in a mature form
and needs an incremental methodology-level update.

Methodology-level additions may include:
- new source-of-truth files
- new recovery rules
- new reporting / projection rules
- new contextJSON generation rules
- context restore policy (Fast/Full + triggers + command mapping)
- goal traceability layer (Requirement IDs + AI task gate)
- response-format execution gates

Mandatory update behavior:
- transfer only operating-system methodology and process contracts
- do not transfer project-specific product architecture or implementation plans
- apply pointwise updates; avoid full rewrites unless explicitly requested
